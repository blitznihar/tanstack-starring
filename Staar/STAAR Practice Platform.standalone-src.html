TEKS 3.3B LESSON · PART A

Fractions on a Number Line

A number line can be split into equal parts. Each part is a fraction of the whole line from 0 to 1.

WORDS TO KNOW
Numerator — top number
Denominator — equal parts
Equal parts — same size
This line is split into 4 equal parts. The point is at 3/4.
WORKED EXAMPLES
1
Count the equal parts from 0 to 1 — there are 4. So the bottom number (denominator) is 4.
2
Count the jumps from 0 to the point — that's 3 jumps. So the top number (numerator) is 3.
3
Put them together: 3 parts out of 4 equal parts = 3/4.
You've got this, Maya — fractions are just equal pieces of a whole! 💪
